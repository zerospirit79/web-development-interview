
1. В чем заключается суть ключевого слова global? Когда его применение целесообразно?

Используется для создания глобальной переменной. 

2. Какие суперглобальные переменные вы знаете? 

boolean – логический;
integer – целочисленный;
float – числа с плавающей точкой;
string – строка;
array – массив;
object – объект;
resource – особый тип данных, который содержит ссылки на внешние ресурсы (файлы, БД и т.д.);
NULL – пустота, отсутствие какого-либо значения.

3. Когда нужно использовать закрывающий дескриптор “?>“? 

Его нужно использовать всегда, это закрывающий дескриптор для скрипта PHP.
